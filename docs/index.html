ataqv: ATAC-seq QC and visualization

About

ataqv is a toolkit for measuring and visualizing your ATAC-seq results. We wrote it to help us understand how well our ATAC-seq assays had worked, and to make it easier to spot differences that might be caused by library prep or sequencing.

Documentation

The online documentation is currently all in the GitHub project README.

Example

We've generated an instance of the ataqv results viewer, stocked with data from the original ATAC-seq paper. You can check it out here, and if you'd like to compare it to your own results, you can either download each file individually from the demo, or clone the ataqv Git repository and add to your mkarv command the JSON files you want from the docs/demo/data/ subdirectory.
