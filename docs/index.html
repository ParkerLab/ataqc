Example

We've generated an instance of the ataqc results viewer, stocked with public data from several papers that incorporated ATAC-seq assays. You can check it out here, and if you'd like to compare the data here with your own, you can either download each file individually from the demo, or clone the ataqc Git repository and add to your mkarv command the JSON files you want from the docs/demo/data/ subdirectory.
